Hi,

the latest version of the specification document of Catena (v3.2) can
be found here:

Paper:
https://www.uni-weimar.de/de/medien/professuren/mediensicherheit/researc
h/catena/

Reference Implementation:
https://github.com/medsec/catena

Important changes:
  - updated test vectors
  - reference implementation now supports client-independent update for
    keyed password hashing and keyed server relief
  - introducing Catena-Axungia (automated benchmark tool to find
    optimal cost parameters)
  - introducing Catena-Variants (implementation of Catena that allows
    extensions and modifications)
